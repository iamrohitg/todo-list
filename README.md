## To-Do List Web Application
This is a simple to-do list web application that allows users to keep track of their tasks and priorities.

![to-do_list_ss](https://user-images.githubusercontent.com/88309940/211007112-b94ffc26-6411-4324-9246-5aa3ea44e8f4.png)

## Features
Add new tasks with a title and optional description

Mark tasks as complete

Edit or delete existing tasks

View all tasks or filter by status (complete or incomplete)

## Technologies Used

HTML, CSS, Bootstrap, JavaScript, Express, EJS.
